Ruby Australia Website

Required Environment / Minimum Setup
------------------------------------

* Install the Ruby version: 2.3.0.
* Install PostgreSQL.
* Install Bundler.
* `bundle install`
* `cp config/database.yml.example config/database.yml` and modify as needed.
* `rake db:setup`
